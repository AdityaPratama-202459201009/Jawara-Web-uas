# JAWARA (Belajar Bahasa Jawa)

Aplikasi pembelajaran Bahasa Jawa berbasis web menggunakan PHP Native MVC dan SQLite.

## Fitur

- Dashboard
- Kategori
- Kosakata
- Belajar
- Kuis
- Tentang

## Teknologi

- PHP Native
- SQLite
- Bootstrap
- MVC

## Cara Menjalankan

1. Clone repository
2. Simpan di folder htdocs
3. Jalankan Apache
4. Buka

http://localhost/jawara/public
